- API: POST /v2/api/products
- Renders result via renderProduct()
- 2 examples: minimal create, create with description

**update.go** — newUpdateCmd():
- Args: cobra.ExactArgs(1)
- Flags: --name (optional), --description (optional)
- Body: only include changed flags via Flags().Changed()
- Error if no fields specified: "no fields specified to update"
- API: PUT /v2/api/products/{id}
- Renders result via renderProduct()
- 2 examples: update name, update multiple fields

**delete.go** — newDeleteCmd():
- Clone directly from cmd/subscribers/delete.go
- Change: "subscriber" -> "product", API path -> "/v2/api/products/"+id
- Uses resource.ConfirmDelete("product", id, yes, cmd.Output.IsJSON())
- Quiet mode suppresses output
- 2 examples: delete with confirmation, delete with --yes
