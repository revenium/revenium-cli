**tools.go** — Parent command + helpers:
- Package: `tools`
- Cmd Use: "tools", Short: "Manage tools"
- Example: list, get, create examples using "revenium tools"
- tableDef: Headers `["ID", "Name", "Type", "Provider", "Enabled"]`, StatusColumn: -1 (no status column; "Enabled" is a boolean, not a status string)
- toRows: extract "id", "name", "toolType", "toolProvider", "enabled" from each tool map. For "enabled", format boolean as "true"/"false" string.
- str(): identical helper (each package has its own)
- boolStr(): new helper -- `if v, ok := m[key]; ok && v != nil { return fmt.Sprint(v) }; return ""` (same as str, booleans Sprint to "true"/"false")
- renderTool(): single-row render using same 5 columns

**list.go** — newListCmd():
- API: GET /v2/api/tools
- Empty: "No tools found." in text, empty array in JSON
- 2 examples: basic list, list with --json

**get.go** — newGetCmd():
- Args: cobra.ExactArgs(1)
- API: GET /v2/api/tools/{id}
- Renders single tool via renderTool()
- 2 examples: basic get, get with --json

**create.go** — newCreateCmd():
- Required flags: --name (string), --tool-id (string), --tool-type (string)
- Optional flags: --description (string), --tool-provider (string), --enabled (bool, default true)
- Body: always include "name", "toolId", "toolType"; include optional fields only if Flags().Changed()
- For --enabled: use BoolVar with default true. Only include in body if Flags().Changed("enabled") to avoid always sending it.
- API: POST /v2/api/tools
- Renders result via renderTool()
- 2 examples: minimal create (required only), create with all fields

**update.go** — newUpdateCmd():
- Args: cobra.ExactArgs(1)
- All flags optional: --name, --tool-id, --tool-type, --description, --tool-provider, --enabled (bool)
- Body: only include changed flags via Flags().Changed()
- Error if no fields specified: "no fields specified to update"
- API: PUT /v2/api/tools/{id}
- Renders result via renderTool()
- 2 examples: update name, update type and provider

**delete.go** — newDeleteCmd():
- Clone from cmd/subscribers/delete.go
- Change: "subscriber" -> "tool", API path -> "/v2/api/tools/"+id
- Uses resource.ConfirmDelete("tool", id, yes, cmd.Output.IsJSON())
- 2 examples: delete with confirmation, delete with --yes
