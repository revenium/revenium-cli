  <files>cmd/sources/sources.go, cmd/sources/create.go, cmd/sources/create_test.go, cmd/sources/update.go, cmd/sources/update_test.go, cmd/sources/delete.go, cmd/sources/delete_test.go</files>
  <behavior>
    - Create with --name "My Source" --type "AI": sends POST /v2/api/sources with {"name":"My Source","type":"AI"}, renders result
    - Create with --description: includes description in body
    - Create missing --name: returns error (required flag)
    - Create missing --type: returns error (required flag)
    - Update with --name only: sends PUT /v2/api/sources/{id} with {"name":"..."} only (partial update)
    - Update with no flags: returns "no fields specified to update" error
    - Delete with --yes: calls DELETE /v2/api/sources/{id} without prompting, prints "Deleted source {id}."
    - Delete in JSON mode: skips prompt
    - Delete with quiet mode: suppresses confirmation message
  </behavior>
  <action>
    1. Create `cmd/sources/create.go`:
       - `func newCreateCmd() *cobra.Command` with Use: "create", Short, 2 examples
       - Flags: `--name` (string, required via MarkFlagRequired), `--type` (string, required), `--description` (string, optional)
       - Long flags only — no shorthand (per locked decision)
       - RunE: build `map[string]interface{}` body from flags, call `cmd.APIClient.Do(ctx, "POST", "/v2/api/sources", body, &result)`, call `renderSource(result)`

    2. Create `cmd/sources/update.go`:
       - `func newUpdateCmd() *cobra.Command` with Use: "update <id>", Short, Args: cobra.ExactArgs(1), 2 examples
       - Flags: `--name`, `--type`, `--description` (all optional strings)
       - RunE: build body using `c.Flags().Changed("name")` etc. — only include explicitly set flags
       - If body is empty (len == 0): return `fmt.Errorf("no fields specified to update")`
       - Call `cmd.APIClient.Do(ctx, "PUT", "/v2/api/sources/"+id, body, &result)`, call `renderSource(result)`

    3. Create `cmd/sources/delete.go`:
       - `func newDeleteCmd() *cobra.Command` with Use: "delete <id>", Short, Args: cobra.ExactArgs(1), 2 examples
       - RunE: get `yes` flag via `c.Flags().GetBool("yes")` (inherited from root persistent flag)
       - Call `resource.ConfirmDelete("source", id, yes, cmd.Output.IsJSON())`
       - If not confirmed: return nil (user cancelled)
       - Call `cmd.APIClient.Do(ctx, "DELETE", "/v2/api/sources/"+id, nil, nil)`
       - If not quiet: `fmt.Fprintf(c.OutOrStdout(), "Deleted source %s.\n", id)`

    4. Update `cmd/sources/sources.go` init() to register create, update, delete subcommands.

    5. Create test files using httptest.NewServer pattern:
       - `create_test.go`: test successful create with flags, test missing required flag
       - `update_test.go`: test partial update (only changed flags in request body), test no-flags error
       - `delete_test.go`: test delete with --yes (skips prompt), test delete output message

    Test pattern: Set `cmd.APIClient` to test server client, `cmd.Output` to `NewWithWriter(buf, errBuf, jsonMode, quiet)`, execute command, assert output and HTTP request body.
