API endpoints for alerts:
- List alerts:        GET /v2/api/sources/ai/alert
- Get alert:          GET /v2/api/sources/ai/alert/{id}
- Create alert rule:  POST /v2/api/sources/ai/anomaly (creates anomaly that generates alerts)
- Budget portfolio:   GET /v2/api/ai/alerts/budgets/portfolio
- Budget progress:    GET /v2/api/ai/alerts/{anomalyId}/budget/progress

Budget progress response fields:
  currentValue, remainingBudget, percentUsed, aheadBehind, budgetThreshold, currency
